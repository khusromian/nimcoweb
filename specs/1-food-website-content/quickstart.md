# Quickstart: Food Website

## Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser to view the website
3. Edit content in `index.html` to customize your information
4. Add product images to the `images/` folder
5. Modify styles in `css/styles.css` to adjust the design

## File Structure

- `index.html` - Main website content and structure
- `css/styles.css` - Main stylesheet
- `css/mobile.css` - Mobile-specific styles
- `css/desktop.css` - Desktop-specific styles
- `js/main.js` - JavaScript functionality (if needed)
- `images/` - Product and logo images

## Customization

### Brand Information
- Edit the `<title>` tag in `index.html` for your brand name
- Update the header section with your brand name and tagline

### Products
- Modify the products section in `index.html` with your product details
- Add product images to the `images/` folder and update image paths

### Contact Information
- Update the contact section with your actual phone and WhatsApp details

## Testing
- Test on multiple devices and screen sizes
- Verify all call-to-action buttons work properly
- Check that all content is readable and accessible
